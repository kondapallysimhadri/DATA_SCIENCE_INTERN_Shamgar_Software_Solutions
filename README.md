# 🦾 TASK 1: Prosthetic EMG Signal Analysis

## 📊 Problem Statement
Analyze EMG signals to understand **muscle activity** for prosthetic systems.

## 🔧 Solution
Built an interactive **Streamlit dashboard** to visualize and analyze multi-channel EMG signals.

## ✨ Features
- 📈 Signal visualization over time  
- 📊 Distribution analysis  
- 🔗 Correlation heatmap  
- 🔍 Multi-channel comparison  
- 🧠 Insights generation  

## 🛠️ Tech Stack
`Python` `Pandas` `NumPy` `Matplotlib` `Seaborn` `Streamlit`

## 📸 Output Preview
![EMG Demo](assets/emg.gif)

## 🎯 Result
- Identified muscle activation patterns  
- Compared signal strength across channels  
- Built real-time analysis dashboard

# 🌾 TASK 3: Crop Recommendation System

## 📊 Problem Statement
Recommend the best crop based on **soil nutrients and environmental conditions**.

## 🔧 Solution
Built a **Machine Learning model + Streamlit app** for real-time prediction.

## ✨ Features
- 🤖 Random Forest Classifier  
- 🎛️ User input via sliders  
- ⚡ Instant crop prediction  
- 📊 High accuracy (~100%)  

## 📥 Inputs
- Nitrogen (N), Phosphorus (P), Potassium (K)  
- Temperature, Humidity  
- pH, Rainfall  

## 📤 Output
- Recommended crop (e.g., rice, wheat, mango)

## 🛠️ Tech Stack
`Python` `Pandas` `NumPy` `Scikit-learn` `Streamlit`

## 📸 Output Preview
![Crop Demo](assets/crop.gif)

## 🎯 Result
- Built accurate ML model  
- Deployed interactive prediction system  
- Solved real-world agriculture problem
