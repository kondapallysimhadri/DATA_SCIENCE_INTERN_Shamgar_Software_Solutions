# EEG Brainwave Analysis Dashboard

An interactive Streamlit dashboard to analyze EEG brainwave data and understand cognitive states like attention, meditation, and brain activity patterns.

---

## Project Overview

This project visualizes EEG data to explore different brainwave frequencies:

- Delta
- Theta
- Alpha1 & Alpha2
- Beta1 & Beta2
- Gamma1 & Gamma2

It helps in:

- Understanding brain activity patterns
- Analyzing attention and meditation levels
- Identifying dominant brainwaves

---

## Features

- Dataset preview
- Brainwave distribution visualization
- Brainwave comparison (Delta → Gamma)
- Attention vs Meditation analysis
- Label distribution
- Correlation heatmap
- Dominant brainwave detection
- Top 3 brainwaves
- Key metrics dashboard

---

## Brainwave Explanation

| Brainwave | Meaning |
|----------|--------|
| Delta | Deep sleep |
| Theta | Relaxation / drowsiness |
| Alpha | Calmness / meditation |
| Beta | Active thinking / focus |
| Gamma | High-level processing |
